# PARENT_PAGE_ENGAGEMENT_IDEAS.md

Phase 3D v3-r1 추가 요구사항(학부모 페이지 체류 시간·설득력 강화) 대응 문서.
Claude가 제안하는 아이디어를 정리하고, 실제 반영 여부를 표시한다.

⚠ 원칙: 어떤 아이디어도 학생용 게임형 지표를 학부모 화면에 직접 노출하지 않는다.
학부모에게는 객관적 데이터(테스트 결과/출결/대학추천/학습 리포트)를 "성장 리포트"
형태로만 보여준다.

---

## 실제 코드에 반영한 항목

1. **탭 구조(테스트/출결/목표대학/리포트)** — 한 화면 안에서 눌러볼 거리를 늘리기 위해
   ParentGrowthReport.tsx를 4개 탭으로 재구성. (`src/pages/parent/ParentGrowthReport.tsx`)
2. **기간 필터(이번 달/최근 3개월/전체)** — 테스트·출결 탭에 적용, 스스로 원하는 범위를
   좁혀볼 수 있게 함.
3. **점수 변화 추이 그래프(SVG 선 그래프)** — 최근 5회 테스트 % 점수를 순수 SVG로 시각화
   (외부 차트 라이브러리 없이 구현, `TrendSparkline` 컴포넌트).
4. **과목별 보완 필요도 막대 그래프** — exam.subject 기준 평균 % 오름차순 정렬(`HBar`).
5. **IF 이유 3종 비율 스택 막대** — `getIfCumulativeSummary()`(기존 studentIfRecord.ts
   함수)를 재사용해 계산 실수/개념 부족/시간 부족 비율을 시각화(`ReasonRatioStack`).
6. **시험 상세 클릭 → 모달** — 카드 클릭 시 실제점수/반평균/최고점 막대 비교 + 놓친 문항
   번호 목록을 보여주는 상세 모달(`TestDetailModal`). 시험지별 그래프 표현 강화 요구사항과
   함께 처리.
7. **자녀 선택(다자녀 지원)** — 기존 ParentHome.tsx 패턴을 재사용해 탭 상단에 배치.
8. **홈 화면 "이번 달 변화" 요약 카드** — 성장 리포트 상단에 테스트 기록 건수/최근 한 달
   평균/보완 필요 과목 3개 요약 카드.
9. **주간/월간 학습 리포트** — 실제 출결·숙제 완료율·테스트 평균을 최근 7일/30일 기준으로
   집계(가짜 데이터 아님 — 기존 AttendanceContext/HomeworkContext/AssessmentContext에서
   직접 계산).
10. **선생님 공개 코멘트 기능 신규 구축** — `src/lib/parentComments.ts` 신규 모듈 +
    `TeacherStudentDetail.tsx`에 작성 UI 추가. 상담 기록(내부용) 원문이 아니라 선생님이
    학부모용으로 직접 다시 쓴 문장만 저장·노출한다.
11. **목표대학 탭에서 기존 화면으로 연결** — `/parent/target-summary`(이미 완성되어 있던
    화면)로 자연스럽게 이어지는 진입 카드로 처리해 로직 중복을 피함.
12. **문장형 해석** — "최근 테스트 점수가 N%p 올랐습니다"처럼 숫자 대신 읽기 쉬운 문장으로
    변화를 설명.
13. **ParentHome.tsx 성장 리포트 진입 카드에서 Tier/Emblem/SP 표현 완전 제거** — 대신
    "테스트 변화 · 출결 흐름 · 대학추천" 문구로 교체하고, `useGrowth()`/`growthData.ts`
    의존을 이 파일에서 아예 제거함.

## 아직 반영하지 못한 항목

1. **최근 3회/최근 5회 "비교" 전용 뷰** — 현재는 "추이 그래프"(연속 값)로만 보여주고,
   두 시점을 나란히 놓고 비교하는 전용 UI(예: 이전 시험 vs 이번 시험 막대를 좌우로 배치)는
   아직 만들지 않았다. 다음 라운드에서 `TestDetailModal`에 "직전 시험과 비교" 섹션을
   추가하면 좋을 것 같다.
2. **월간 리포트의 "전월 대비" 비교** — 이번 달 수치만 보여주고 있고, 지난달과 비교한
   변화량(%p, 증감 화살표)은 아직 계산하지 않았다.
3. **수업자료 열람 현황을 성장 리포트 안에 통합** — 현재 수업자료는 학부모 홈에서 별도
   섹션으로만 노출되고(v2), 성장 리포트의 "리포트" 탭 안에는 아직 연결하지 않았다.
4. **문항별 정오표를 더 시각적으로(그리드/색상 매트릭스)** — 현재는 오답 문항 번호를
   칩으로 나열하는 수준이다. 정답 문항까지 포함한 전체 문항 그리드(초록/빨강 배경)로
   확장하면 더 한눈에 들어올 것이다.

## 추후 Phase에서 반영할 항목

1. **알림/푸시 기반 재방문 유도** — "새 테스트 결과가 공개되었습니다" 같은 알림으로
   학부모가 다시 접속하게 만드는 흐름. 현재 알림 시스템은 출결 알림에만 쓰이고 있어
   확장이 필요하다.
2. **학기 단위 성장 곡선(1년 전체 추이)** — 지금은 최근 5회/기간 필터 단위지만, 장기
   재원 학생의 경우 "1학기 대비 2학기" 같은 학기 단위 비교가 학부모 설득에 더 강력할 수
   있다.
3. **PDF/이미지로 리포트 내보내기(공유하기)** — 학부모가 배우자나 다른 보호자에게 리포트를
   공유하고 싶을 수 있다. 현재는 화면 안에서만 확인 가능하다.
4. **목표대학 탭의 실제 변화 이력(시계열)** — 현재는 "최신 상태"만 보여주는 기존
   ParentTargetSummary.tsx로 연결한다. "지난달 대비 어떻게 바뀌었는지" 변화 자체를
   추적하려면 University Analysis 쪽에 스냅샷 이력 저장 기능이 먼저 필요하다.
5. **선생님 공개 코멘트에 학부모 "확인했습니다" 반응 기능** — 현재는 읽기 전용이다.
   학부모가 코멘트를 확인했는지 선생님이 알 수 있게 하면 상호작용이 늘어날 수 있지만,
   "학부모는 확인자다 — 입력 기능 금지" 원칙과 충돌할 수 있어 신중한 검토가 필요하다.

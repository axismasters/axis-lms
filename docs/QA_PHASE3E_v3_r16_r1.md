### A9 상세 — tsc 스텁 하네스

`npm install`이 이 샌드박스에서 `E403 host_not_allowed`(네트워크 정책)로 실패해 실제
`tsc -b`/`vite build`를 실행할 수 없다. 오프라인 스텁으로 대체 검증했다(이번 Phase에서
`Fragment` named export 누락을 발견해 스텁에 추가 — 프로젝트 전체 스코프 오류가
73건→72건으로 줄었다, 실제 코드 변경 아님).

- **수정 파일 5종 스코프**: 오류 0건.
- **전체 프로젝트 스코프**: 72건 — 전부 이번 변경과 무관한 기존 파일(`FinanceRefunds.tsx`
  등)에서 발생.
- **"npm run build 통과"를 이 세션에서 재확인했다고 주장하지 않는다.** 최종 확인은
  GitHub Actions에서 이루어져야 한다.

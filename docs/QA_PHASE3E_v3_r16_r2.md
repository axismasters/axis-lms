### A10/A11 상세 — tsc 스텁 하네스

`npm ci`가 이 샌드박스에서 `E403`으로 실패해 실제 `tsc -b`/`vite build`를 실행할 수
없다. 오프라인 스텁으로 대체 검증했다.

- **수정+신규 파일 6종 스코프**: 오류 0건(신규 `universityCounselingSummary.ts`는 첫
  시도에 오류 없이 통과했다 — 기존 `universityPayloadAdapter.ts`의 export 타입을
  그대로 재사용했기 때문).

  Phase들에서 이미 확인된 것과 동일한 스텁 근사 한계).
- **"npm run build 통과"를 이 세션에서 재확인했다고 주장하지 않는다.** 최종 확인은
  GitHub Actions에서 이루어져야 한다.
